# 封装ChatGPT的会话服务
## ChatMe采蜜

不是使用sessionToken那种，因为sessionToken太难搞了，翻墙模式下sessionToken拿到也运行不起来。

本程序使用openai官方的apikey，配置上之后开箱可用。
